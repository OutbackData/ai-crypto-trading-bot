# AI Crypto Trading Bot (LLM Powered)

AI-driven BTC/USDT trading bot that combines technical indicators with an LLM decision engine.

## Features

- Real or paper trading
- Binance.US integration
- Technical indicators (RSI, MACD, ATR)
- LLM-based decision making
- Live PnL dashboard
- Automated trade execution

## Stack

- Python
- CCXT
- Ollama
- Llama3
- Pandas
- NumPy

## Architecture

Market Data → Indicators → AI Decision → Trade Execution → PnL Dashboard
